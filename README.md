# About

**Features**
- 0. 구글 로그인 
- 1. 집안일 생성 
- 2. 집안일 조회
- 3. 집안일 삭제  
- 4. 그룹 생성 , 카카오톡 초대 , 딥링크  
- 5. 프로필 수정 
- 6. 집안일 규칙 추가


**Technology Stack**
- Tools : Android Studio Bublebee
- Language : Kotlin
- Architecture Pattern : MVVM Pattern
- Android Architecture Components(AAC)
  - Flow
  - ViewModel
  - Coroutine
  - Data Binding
- Naivgation Conponponent
- OKHTTP
- RETROFIT
- MOSHI
- SERIALIZATION
- KAKAO_SDK
- GLIDE
- FIREBASK_BOM
- TIMBER
